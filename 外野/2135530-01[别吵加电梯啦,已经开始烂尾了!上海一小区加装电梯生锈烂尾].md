近日，静安区江宁路街道市民徐先生通过解放日报&amp;#xB7;上观新闻互动平台留言求助，称自己居住的老小区13号楼新加装了一部电梯，但有很多质量问题，自2022年初建设完毕烂尾至今，无法投入使用，希望能早日整改好投入运行。选材偷工减料4月25日，记者来到徐先生居住的常德路519弄又一邨小区。进入正门便可看到小区深处的13号楼新加装的电梯，外观呈米黄色，共7层，已经建设完毕。但走近一看，这个电梯并未开放使用，独立铁门已被一把U型锁锁住，按键上的显示屏也处于黑屏状态。铁门上方有一块挡雨的[url=]玻璃[/url]檐，下方支撑的钢架已有黄褐色锈迹。在电梯主体建筑与13号楼连接区域旁，钢梁上的锈色也十分明显。[url=]<img src="https://k.sinaimg.cn/n/spider20230508/250/w600h450/20230508/6066-e5486f0cc3270af6a5643d1d3e13d127.jpg/w700d1q75cms.jpg?by=cms_fixed_width" id="aimg_t0fBo" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)<strong>△常德路519弄又一邨小区。</strong>[/url]从单元门进入13号楼，可以看到电梯采用平层入户的形式，电梯门外的区域已被不少家庭用来放置了杂物。电梯门对面的墙壁多有破损，透过数十厘米见方的大洞可以看到里面的钢结构。居民所说的质量问题，有一部分便与钢结构有关。记者了解到，2021年上半年，静安区常德路519弄小区进行了业主大会征询，全票通过了13号楼加装电梯项目。经一系列程序后，项目由上海屹宏电梯科技发展有限公司代建，上海成今建设工程有限公司总包，于2021年8月18日开工，计划2021年12月21日竣工，实际完工于2022年初。“我上班的时候无意间看到他们钢梁来货了，用手机拍了一下这个钢梁的规格，后来发现与电梯施工图纸对不上。”2021年下半年，在13号楼电梯加装项目施工过程中，楼里居民徐先生偶然间拍摄到了入场[url=]钢材[/url]的合格证，上面的规格一栏写着“200*200*5.5*6000”，即厚度5.5毫米。电梯施工进入尾声时，代建公司联系居民称“可以试用了”。居民便向代建公司要了图纸，准备进行简单的检查。借着这个机会，徐先生在施工图纸上仔细查找关于钢材的要求，发现图纸的材料表注明了钢柱、主梁、雨篷梁均采用8毫米厚度规格的Q235B钢，最薄的次梁、导轨梁、构造柱也要求达到6毫米厚度。[url=]<img src="https://k.sinaimg.cn/n/spider20230508/299/w535h564/20230508/8dca-ddddd43f3b3e683ba85ef21dac4f9052.png/w700d1q75cms.jpg?by=cms_fixed_width" id="aimg_ySOKO" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)<strong>△钢柱、主梁、雨篷梁均应采用8毫米厚度规格的Q235B钢。</strong>[/url]此外，电梯房还有比较严重的漏雨现象，居民猜测可能是未按图纸设计安装雨篷的原因，坡道、栏杆等等也都没有安装好。一名曾在江南造船厂工作过的居民试用时还发现，电梯主体结构的很多焊接点质量不高。
[url=]<img src="https://k.sinaimg.cn/n/spider20230508/322/w482h640/20230508/cb63-5f00bee75c46cc490baf886d815cd6d3.jpg/w700d1q75cms.jpg?by=cms_fixed_width" id="aimg_L5UoF" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)[/url][url=]<img src="https://k.sinaimg.cn/n/spider20230508/250/w600h450/20230508/a79d-45607516862d98a27b1e58999f75736f.jpg/w700d1q75cms.jpg?by=cms_fixed_width" id="aimg_CURTZ" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)[/url]
种种迹象让13号楼居民们对电梯总体质量产生了深深的怀疑。检测与验收结果不同2022年下半年，按照居民们的要求，代建公司委托上海市房屋建筑设计院有限公司房屋质量检测站（下称“上房院”）对电梯部分材料的施工质量进行了专项检测，主要复核了电梯建筑结构布置情况，检测了电梯底板混凝土强度，以及主体结构截面尺寸和强度。今年3月6日，上房院出具了专项检测咨询报告。虽然检测时加梯工程已经施工完毕，现场没有条件对所有构件节点进行全面普查，但抽测仍然可以发现，徐先生和其他居民们的怀疑和质疑基本属实。检测报告提到，电梯的主、次钢梁截面尺寸均与图纸不符，梁柱节点做法也与图纸不符。本该使用8毫米厚度钢材的主梁，经检测厚度在5.42毫米至5.72毫米之间，差距很大。检测报告提到，钢梁节点焊缝表面较多锈蚀，防锈涂料剥落。但2021年12月15日签字盖章的《压型金属板检验批质量验收记录》对于钢结构的10项验收项目检查结果全部合格，监理单位验收结论为“符合要求”。上房院检测发现，电梯基坑底板的混凝土强度经抽样检测也不满足原设计要求的C35级别，抗压强度推定值仅为26.1MPa，相当于C25级别左右的强度，与图纸要求标准相差甚远。对此，徐先生回忆，图纸设计公司上海华铤建筑设计有限公司曾向居民们推测，可能是混凝土放置过久，导致发生轻微凝固现象，使浇筑后强度变低。而在电梯加装项目中，施工记录表及质量验收记录却根本未发现这些问题。一份2021年8月30日签字盖章的《混凝土原材料及配合比设计检验批质量验收记录表》中，水泥进场检验、外加剂质量及应用、配合比设计、粗细骨料的质量、拌制混凝土用水等项目，包括“混凝土强度等级”，验收记录均是“符合”，监理单位验收结论也是“符合要求”。记者按施工铭牌上的电话尝试联系项目经理和监理求证，项目经理电话号码已停机，监理电话号码的持有人称记者“找错人了”。变重建为加固，居民不接受针对居民反映的主要问题，记者联系了静安区有关部门。江宁路街道工作人员回复称，经过核实，居民反映的情况属实。代建公司此前已要求施工方进行了撤场、处罚、终止合作，对监理单位进行了严重警告，并对当事监理负责人进行了停职调岗建议，对工程负责人进行了停职处分，解除了劳动关系。自试用起，居民们和代建单位上海屹宏电梯科技发展有限公司不断反映质量问题，多次协商沟通。就如何整改，居民与代建公司之间有不同的意见。徐先生回忆称，最初，代建公司称愿意拆除重建，而且经征询居民意见后，拆除重建的计划也得到了通过。但不久后，代建公司又否认了之前的说法，表示只能进行加固处理，理由是“只是有一些小问题，做一些改正就可以了，这样也能早点投入使用。”[url=]<img

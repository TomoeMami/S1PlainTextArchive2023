目前发现整合包有两个图形插件存在问题：

1、"Hide Offline Label"，隐藏画面左上角的离线提示。整合包里的这个版本失效，而且会导致无法显示设置界面。下载最新社区图形插件，勾选对应的新版本可解决。

2、"60FPS"，顾名思义。整合包里的这个版本会导致QTE卡住不缩圈，而最新的社区版本却会导致过场动画镜头不运动。我尝试把旧版插件的一些地址替换成最新版的，解决了QTE的问题，但还需要进一步测试。具体方法如下：

1）使用记事本打开Cemu目录下的graphicPacks\XenobladeChroniclesX\JP\Mods\JP-60FPS\patch_60fps_qte.asm

2）找到位置 [V48J] ######################################################## moduleMatches = 0x7672271D ; 1.0.2J复制代码
将该位置下方的代码 ; menu::MenuButtonChallenge::setup 0x02ACAA28 = lis r7, _setup@ha 0x02ACAA30 = lfs f0, _setup@l(r7)  ; menu::MenuButtonChallenge::move 0x02AD20A8 = bla _justFrame1 0x02AD20C4 = bla _justFrame2复制代码
替换为 ; menu::MenuButtonChallenge::setup 0x02ACAA38 = lis r7, _setup@ha 0x02ACAA40 = lfs f0, _setup@l(r7)  ; menu::MenuButtonChallenge::move 0x02ACAD10 = bla _justFrame1 0x02ACAD2C = bla _justFrame2复制代码

3）保存关闭，然后打开Cemu勾选Xenoblade Chronicles X/JP/Mods/60FPS

另外，个人感觉整合包的"Brightness"插件比一键下载的社区最新版要更加自然。新插件似乎通过压低全局曝光来解决白天过曝的问题（rules.txt里的exposure参数），导致标题画面和捏人默认背景过暗，与实机差距很大。

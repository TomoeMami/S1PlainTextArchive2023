《刺客信条》系列日本背景新作《刺客信条：代号 RED》预告公开，由育碧魁北克（曾开发《刺客信条：奥德赛》）负责开发。 http://t.cn/A6S8bknc ​​​

【传闻：多款《刺客信条》即将登场】知名消息人士Tom Henderson表示，育碧会在本周末的前瞻会上一次性放出多款《刺客信条》的情报，以下是他在几个月来搜集的信息：

·《刺客信条 幻景》：故事设定在巴格达，主角为年轻时期的巴辛姆，2023年推出

·《刺客信条 Project Red》：《刺客信条 无限》的一部分，以日本为背景，概念图为女性武士，但玩家可能可以选择性别，2024年推出

·《刺客信条 女巫（Hexe）》：《刺客信条 无限》的一部分，以十六世纪欧洲猎巫为背景，为AC系列最黑暗的故事，2026年推出

·《刺客信条 Project Jade》：中国背景的手机游戏

·《刺客信条 英灵殿》新DLC

·《刺客信条 Project Nexus》：《刺客信条》系列的VR项目，但不清楚是否会在周末公开

据知名游戏记者Jeff Grubb爆料，《刺客信条：无限》将于9月的Forward发布会上公开，游戏将是一个不断更新的平台，会有多种设定或游戏。本作首发时将有两个设定，一种是以日本为背景、扮演忍者的“Project Red”，另一种则尚未知晓。 ​​​

在育碧财报公布后，外媒kotaku发文表示，代号为“赤红（Project Red）”的新开放世界《刺客信条》游戏正在开发，而记者Jason Schreier表示，该项目将是《刺客信条：无限》的一部分。

而kotaku从育碧发言人处获悉，无论“赤红（Project Red）”是独立游戏还是《刺客信条：无限》中的一部分，该项目的故事背景都将设定在亚洲。

“赤红（Project Red）”和《无限》可能不是育碧目前唯一正在开发的新《刺客信条》游戏。Kotaku报道称，还有另一款代号为“Project Rift”的新游戏，据报道，这是一款规模较小的《刺客信条》游戏，开发之初曾作为《刺客信条：英灵殿》的DLC，后来成为独立游戏。

消息人士告诉Kotaku，据报道，这款游戏以巴格达为背景，但该作目前在开发过程中“面临着新的挑战”。Kotaku猜测，这款游戏就是育碧近期推迟到“规模较小且尚未公布”的游戏。

男性武士是一位非洲难民，并习得了刺客教义，这位角色可能是以日本历史上第一位黑人武士弥助为原型。

弥助活跃于日本安土桃山时代，是战国大名织田信长的一位来自非洲的黑人家臣。

弥助是传教士范礼安从印度带来的奴隶，担任膂力劳动者，所以游戏中这位武士扮演类似角色，力大无穷且壮如蛮牛。

女性忍者的原型是谁目前尚不清楚，唯一信息是她比武士更加灵敏。

不过可以确定的是：两位角色将会拥有完全不同的特点和武器，使彼此独一无二。

游戏特点：

《刺客信条 代号红色》是一款主打潜行，并且借鉴《细胞分裂》系列玩法的游戏。

例如，玩家能够利用阴影隐藏自己，并可以熄灭火烛潜藏在黑暗之中（类似《细胞分裂》打碎探照灯）

当然，敌人的尸体也可以被搬运并转移到难以发现的地方。

游戏的另一大特色是可破坏的环境物品，玩家可以用手中的道具打破游戏世界中任何可能被打破的东西。

例如，屏门可以被划破或砸开，这为《刺客信条 代号红色》增添了更多乐趣，玩家一定会喜欢。
